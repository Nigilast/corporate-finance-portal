Коэффициентный анализ, формула Дюпона, модель Альтмана (Z-score)

Проблемный вопрос

Как за 15 минут определить, здорова ли компания финансово? Какие 5 коэффициентов расскажут больше, чем 100 страниц отчётности?

2. Ключевые формулы

Ликвидность
$$\text{Current Ratio} = \frac{\text{Оборотные активы}}{\text{Краткосрочные обязательства}}$$ $$\text{Quick Ratio} = \frac{\text{Оборотные активы} - \text{Запасы}}{\text{Краткосрочные обязательства}}$$
Рентабельность
$$ROE = \frac{\text{Чистая прибыль}}{\text{Собственный капитал}}$$ $$ROIC = \frac{NOPAT}{\text{Инвестированный капитал}}$$
Финансовая устойчивость
$$\text{Interest Coverage} = \frac{EBIT}{\text{Процентные расходы}}$$ $$\text{Net Debt/EBITDA} = \frac{\text{Долг} - \text{Денежные средства}}{EBITDA}$$

3. Формула Дюпона (5-факторная)

5-факторная модель DuPont
$$ROE = \frac{NI}{EBT} \times \frac{EBT}{EBIT} \times \frac{EBIT}{Sales} \times \frac{Sales}{Assets} \times \frac{Assets}{Equity}$$

= Налоговое бремя × Процентное бремя × Операционная маржа × Оборачиваемость × Леверидж

Практическая ценность DuPont

Позволяет определить источник высокого/низкого ROE. Например: высокий ROE банка = леверидж (7–10x), а ритейлера = оборачиваемость (2–3x).

4. Модель Альтмана (Z-score)

Z-score Альтмана (для публичных компаний)
$$Z = 1{,}2 X_1 + 1{,}4 X_2 + 3{,}3 X_3 + 0{,}6 X_4 + 1{,}0 X_5$$

где:

  • $X_1$ = Оборотный капитал / Активы
  • $X_2$ = Нераспределённая прибыль / Активы
  • $X_3$ = EBIT / Активы
  • $X_4$ = Рыночная капитализация / Долг
  • $X_5$ = Выручка / Активы
Интерпретация Z-score
  • $Z > 2{,}99$ — безопасная зона (низкая вероятность банкротства)
  • $1{,}81 < Z < 2{,}99$ — серая зона (неопределённость)
  • $Z < 1{,}81$ — зона бедствия (высокая вероятность банкротства)
Ограничения для России

Модель разработана на данных американских компаний 1960-х. Для российских компаний Z-score может давать искажения: иная структура баланса, высокий леверидж банков, специфика отчётности. Рекомендуется использовать совместно с отраслевым анализом.

Контрольные вопросы

Какие 5 групп финансовых коэффициентов используются в анализе?

1) Ликвидность, 2) Рентабельность, 3) Деловая активность (оборачиваемость), 4) Финансовая устойчивость (структура капитала), 5) Рыночные показатели (P/E, EV/EBITDA).

Как формула Дюпона помогает диагностировать проблемы?

Раскладывает ROE на 5 факторов, позволяя определить конкретный источник проблемы: низкая маржа → ценовая/затратная проблема; низкая оборачиваемость → неэффективное использование активов; высокий леверидж → финансовый риск.

Что означает Z-score = 2.0 по модели Альтмана?

Компания в «серой зоне» (1.81–2.99): вероятность банкротства не высока, но и не исключена. Требуется дополнительный анализ: тренд Z-score, отраслевое сравнение, качество активов.
