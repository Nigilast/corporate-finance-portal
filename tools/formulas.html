Формульный справочник

Все ключевые формулы курса в одном месте

1. Временная стоимость денег

Наращение (FV)
$$FV = PV \times (1 + r)^n$$
Дисконтирование (PV)
$$PV = \frac{FV}{(1 + r)^n}$$
Аннуитет (PV)
$$PV_{\text{ann}} = PMT \times \frac{1 - (1+r)^{-n}}{r}$$
Перпетуитет
$$PV = \frac{PMT}{r}$$
Растущий перпетуитет (Гордон)
$$PV = \frac{PMT}{r - g}$$
Формула Фишера
$$(1 + r_{\text{ном}}) = (1 + r_{\text{реал}}) \times (1 + \pi)$$
EAR
$$EAR = \left(1 + \frac{r}{m}\right)^m - 1$$

2. Риск и доходность

Ожидаемая доходность
$$E(r) = \sum_{i} p_i \times r_i$$
Дисперсия
$$\sigma^2 = \sum_{i} p_i \times (r_i - E(r))^2$$
Риск портфеля (2 актива)
$$\sigma_p = \sqrt{w_1^2\sigma_1^2 + w_2^2\sigma_2^2 + 2w_1w_2\sigma_1\sigma_2\rho_{12}}$$
CAPM
$$k_e = r_f + \beta \times (r_m - r_f)$$
Бета-коэффициент
$$\beta_i = \frac{Cov(r_i, r_m)}{\sigma_m^2}$$
Коэффициент Шарпа
$$S = \frac{E(r_p) - r_f}{\sigma_p}$$

3. Инвестиционные решения

NPV
$$NPV = \sum_{t=0}^{n} \frac{CF_t}{(1 + r)^t}$$
IRR
$$\sum_{t=0}^{n} \frac{CF_t}{(1 + IRR)^t} = 0$$
PI
$$PI = \frac{PV_{\text{притоков}}}{IC}$$

4. Структура капитала

WACC
$$WACC = w_e \cdot k_e + w_d \cdot k_d \cdot (1-T) + w_p \cdot k_p$$
Стоимость долга (после налогов)
$$k_d^{\text{после}} = k_d^{\text{до}} \times (1-T)$$
Налоговый щит
$$TS = D \times r_d \times T$$
ММ II (с налогами)
$$k_e = k_0 + (k_0 - k_d) \times \frac{D}{E} \times (1-T)$$
Формула Хамады
$$\beta_L = \beta_U \times \left(1 + (1-T) \times \frac{D}{E}\right)$$

5. Рентабельность

ROE
$$ROE = \frac{\text{Чистая прибыль}}{\text{Собственный капитал}}$$
Формула Дюпона (3-факторная)
$$ROE = \frac{NI}{Sales} \times \frac{Sales}{Assets} \times \frac{Assets}{Equity}$$
EVA
$$EVA = NOPAT - WACC \times IC$$

6. Оценка бизнеса

DCF (FCFF)
$$V = \sum_{t=1}^{n} \frac{FCFF_t}{(1+WACC)^t} + \frac{TV}{(1+WACC)^n}$$
Терминальная стоимость
$$TV = \frac{FCFF_{n+1}}{WACC - g}$$
FCFF
$$FCFF = EBIT(1-T) + DA - CAPEX - \Delta WC$$

7. Финансовый анализ

Z-score Альтмана
$$Z = 1{,}2X_1 + 1{,}4X_2 + 3{,}3X_3 + 0{,}6X_4 + 1{,}0X_5$$
